# Logue(로그) 프로젝트 청사진

## 1. 서비스 개요

- **서비스명:** Logue (로그)
- **컨셉:** 매일 영어로 일기를 기록하며 AI의 정교한 첨삭을 통해 원어민의 감각을 익히는 다이어리 학습 서비스.
- **핵심 타겟:** 자기계발에 열정적인 20~40대 글로벌 사용자.
- **기술 스택:**
  - **Frontend:** Next.js (App Router), Tailwind CSS
  - **Backend:** Firebase (Auth, Firestore, Hosting)
  - **AI:** Gemini 1.5
  - **Version Control:** Git & GitHub

## 2. 디자인 및 기능 구현 계획

### 현재 단계: 초기 뼈대 및 개발 환경 구축

1.  **랜딩 페이지 (`/app/page.tsx`):**
    - 2040 타겟에 맞는 미니멀하고 세련된 디자인.
    - 서비스의 핵심 가치(AI 영어 첨삭)를 명확하게 전달하는 Hero 섹션.
    - 주요 기능(스마트 에디터, AI 코치, 개인 보관함)을 소개하는 섹션.
    - 일기 작성 화면으로 이동하는 명확한 Call-to-Action (CTA) 버튼.

2.  **일기 작성 페이지 (`/app/diary/page.tsx`):**
    - 실제 종이 다이어리 같은 감성적인 UI.
    - 영어 전용 입력을 위한 유효성 검사 로직 (한글 입력 시 '완료' 버튼 비활성화).
    - 사용자가 일기를 입력할 수 있는 텍스트 영역.

3.  **개발 환경 설정:**
    - **Firebase 연동:** Firebase Hosting 및 Gemini 통합(MCP) 설정 완료.
    - **GitHub 연동:** 소스 코드 버전 관리를 위해 GitHub 리포지토리 설정 및 연동.
    - **"Gemini CLI" 워크플로우:** 채팅 인터페이스를 통해 AI가 직접 코드를 수정하고 터미널 명령을 수행.
